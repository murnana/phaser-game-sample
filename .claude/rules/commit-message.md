# Commit Message Convention

Use gitmoji prefixes for commit messages, consistent with existing commits.

**Commit messages must be written in Japanese.**

- `:sparkles:` — new feature
- `:bug:` — bug fix
- `:recycle:` — refactor
- `:memo:` — documentation
- `:wrench:` — configuration/tooling

Example: `:sparkles: プレイヤー移動シーンを追加`
