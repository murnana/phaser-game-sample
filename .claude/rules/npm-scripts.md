# npm Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server at http://localhost:8080 (uses `vite/config.dev.mjs`) |
| `npm run build` | Production build to `dist/` (uses `vite/config.prod.mjs`) |

- Do not change the dev server port without updating `CLAUDE.md` and `.devcontainer/devcontainer.json`
- Do not commit the `dist/` directory
