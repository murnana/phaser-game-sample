# コミットメッセージの規則

既存のコミット履歴に合わせて、**gitmoji** プレフィックスを使ってください。

**コミットメッセージは日本語で記述してください。**

| 絵文字コード | 用途 |
|------------|------|
| `:sparkles:` | 新機能の追加 |
| `:bug:` | バグ修正 |
| `:recycle:` | リファクタリング |
| `:memo:` | ドキュメントの追加・更新 |
| `:wrench:` | 設定・ツールの変更 |

**例:**
```
:sparkles: プレイヤー移動シーンを追加
:bug: 背景画像の座標がずれる問題を修正
:memo: developer-guide にルールを追加
```
